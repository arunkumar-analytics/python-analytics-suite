# Python Analytics Suite

## Overview
A command-line data analysis toolkit with 6 useful utilities for everyday data tasks. Built to demonstrate Python fundamentals for data analyst roles.

## Features

| Command | Function | Use Case |
|---------|----------|----------|
| `FF` | First 5 characters | Text extraction |
| `PL` | Profit/Loss calculator | Business finance |
| `PS` | Perfect squares generator | Number theory |
| `XM` | Multiples finder | Math operations |
| `FS` | Fibonacci series | Sequence generation |
| `TS` | Top spender tracker | Expense analysis |
